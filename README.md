## Manual Install
```
$ git clone https://github.com/woat/dotfiles/
--vim--
place .vim + .vimrc in ~/
--zsh--
place .zshrc in ~/
--iterm2--
goto library/preferences -> com.googlecode.iterm2.plist (overwrite + backup if necessary)
iterm2 > preferences > profile(tab) > root 
```
